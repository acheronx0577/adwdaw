✦ Cursor install

Get it
running.

01 Global (recommended)

Skills, commands, and hooks under your user .cursor folder:

.\scripts\install-user-global.ps1

02 Skip marketplace Superpowers

Do not enable marketplace Superpowers — this repo replaces it. If already enabled:

.\scripts\disable-marketplace-superpowers.ps1

03 Per project

Then /mb <your first task> (or /van for full init):

.\scripts\install-project-starter.ps1 -Target "C:\path\to\your-project"

New machine checklist

Clone repo → install-user-global.ps1 → enable marketplace GSAP → skip marketplace Superpowers → optional install-project-starter.ps1 per repo.
